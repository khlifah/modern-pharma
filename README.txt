
# Modern Pharma (شكل Modern Soft - واجهة عربية)

## طريقة التشغيل (XAMPP)
1) افتح phpMyAdmin وأنشئ قاعدة باسم `modern_pharma` أو شغّل سكربت القاعدة:
   - استورد الملف: `db/modern_pharma.sql` (سيبني الجداول والإجراءات، ويضيف مستخدم admin بكلمة `admin123`).
2) انسخ المجلد إلى `C:\xampp\htdocs\modern_pharma`.
3) افتح: `http://localhost/modern_pharma/`
4) بيانات الدخول: **admin / admin123** (غيّرها لاحقاً).

## ماذا جهزت لك الآن
- شاشة دخول + لوحة تشبه Modern Soft (قوائم وأزرار كبيرة).
- تقرير **أرصدة الموردين** مع شاشة عرض وزر **طباعة** بتنسيق نظيف.
- تقرير **كشف حساب مورد** (مبسّط) مع طباعة.
- هيكل قاعدة بيانات يدعم المنتجات، الدفعات، FEFO، فواتير شراء/بيع، الحركات.
- إجراءات مخزنة: `sp_post_purchase` و `sp_post_sale` (FEFO).
- صفحات بادئة لفاتورة شراء/بيع للتوسعة لاحقاً.

> هذا إصدار بداية يمكنك تنزيله وتشغيله فوراً، ثم نضيف بقية النماذج لتصل لمستوى الصور بالكامل.
